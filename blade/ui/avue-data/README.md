## 调试

### 配置密钥
```
参考 [配置密钥](https://www.yuque.com/smallwei/avue-data/xg54owx6dd238lbd)，配置密钥到.npmrc文件
```

### 依赖
```
pnpm install
```
### 运行
```
pnpm dev
```
### 打包
```
pnpm build
```
### 打包单独部署库(index.umd.js)
```
pnpm lib
```
